# Computational Biology Mini Projects

## Project 1: Epitope Prediction
- Retrieved protein sequences from NCBI using Biopython
- Performed hydrophilicity analysis
- Predicted potential B-cell epitopes
- Visualized results using Matplotlib

## Project 2: Evolutionary Analysis
- Compared homologous proteins from Human, Mouse, Rat, Cow, and Pig
- Used Biopython and Pandas for sequence analysis
- Investigated evolutionary conservation patterns

## Tools Used
- Python
- Biopython
- Pandas
- Matplotlib
- Jupyter Notebook
